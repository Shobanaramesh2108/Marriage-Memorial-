# Marriage-Memorial-
Matrimony Website (Flask)
A simple matrimonial website built with Python Flask + SQLite.

Features:
- User registration and login (email + password)
- Profile creation (name, age, religion, city, bio)
- Profile picture upload
- Browse/search profiles
- Basic session management using Flask-Login

Requirements:
- Python 3.8+
- Install dependencies: `pip install -r requirements.txt`

Run:
1. export FLASK_APP=app.py
2. flask run
3. Open http://127.0.0.1:5000

Notes:
- This is a starter project intended for learning and customization.
- Do NOT use in production without adding security, validation, and HTTPS.
